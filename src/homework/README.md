# Homework

This is the homework directory. 

You need an activated virtualenv to run the python CLI applications.
You can activate the virtualenv with the following commands:
* Windows: `source ../../venv/Scripts/activate`
* MacOS / Unix: `source ../../venv/bin/activate`
* Alternative: Use PyCharm's terminal.

Run the CLI Applications:

```commandline
$ python -m appname command --arg value
```
